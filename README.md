# Tetris 
Este proyecto implementa el juego tetris en c++ utilizando qt. Se enfoca principalmente en el uso eficiente de memoria mediante punteros, memoria dinamica y manipulacion de bits

## Estructura del repositorio 
-`desafio1`: carpeta con los archivos del proyecto 
-`Análisis preliminar desafio I.pdf`: archivo con el analisis preeliminar del proyecto
-`INFORME DE ENTREGA PARA EL DESAFIO I.pdf`: informe final del proyecto 
-`README.md`: esta archivo

## Requisitos
-c++
-Qt 
-Compilador compatabile con g++, MinGW, etc.

## Ejecucion
1. Clonar el repositorio
2. Abrir el proyecto en qt
3. Ejecutar el programa
